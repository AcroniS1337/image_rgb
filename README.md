# image_rgb

### Инструкция по установке:
1) клонировать репозиторий git clone https://github.com/AcroniS1337/image_rgb
2) перейди в каталог cd скачанного репозитория(к примеру cd image_rgb)
3) установить зависимости через команду pip install -r requirements.txt
4) Запустить приложение  через команду python main.py
